## Notes

- Agrupar prompts por página é obrigatório; não criar uma página por query.
- As páginas P1 devem ligar de forma explícita para os money pages já existentes.
- O wording de Reddit/AI que interessa aqui é o wording de dor: "não perder chamadas", "sem contratar mais receção", "juntar tudo num só painel".
- Sempre que faltar prova factual, usar secções de critérios, limites e metodologia em vez de claims de performance.
